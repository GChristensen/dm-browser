A standalone Dark Matter instance (runs under the GAE emulation). 

dmbutton-firefox.xpi Firefox addon allows to launch a standalone instance of 
DM Browser directly from Firefox (the DM Browser application will be 
automatically terminated on Firefox exit).
You need to specify a full path of the dm-browser.exe executable in the 
addon options.

darkmatter-firefox.xpi Firefox addon adds chan:// url support to Firefox.
You need to specify a web address of the target Dark Matter server instance
(this also could be a local server).
